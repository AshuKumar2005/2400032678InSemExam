1. Initialize a Git repository
git init

2. Add files to staging
git add .

3. Commit changes with a message
git commit -m "Initial commit"

4. Create a branch feature-login
or
git branch feature-login

5. Make a small change in app.js and commit in the branch
git checkout feature-login
git add app.js
git commit -m "Updated app.js"

6. Switch back to main
git checkout main

7. Change style.css and commit
git add style.css
git commit -m "Updated style.css"

8. Merge feature-login into main
git merge feature-login

9. Handle merge conflicts
git add .
git commit -m "Resolved merge conflicts"

10. Use git diff to view changes
git diff

11. Undo changes using git reset and git checkout
git reset HEAD app.js
or
git reset  -hard HEAD-1

12. Tag the latest commit as v1.0
git tag v1.0
